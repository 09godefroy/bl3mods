Patch Slaughterstar 3000 Boss Drop
==================================

Adding a floor in front of Red Rain and Blue Fire boss locations in the 
Slaughterstar 3000 so that the loot dropped from them will not be lost to
the void.

* Red Rain

* Blue Fire
